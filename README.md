# led-race
Race each other on an LED strip, using an ESP8266

## Arduino libraries

* Regexp
* AsyncHTTPRequest_Generic
* ESP8266WiFi
* WiFiManager

## Running scoreboard

`npm install && npm start` in `pi-scoreboard`
